Название
- Название модуля (Отображается в заголовке страницы и хлебных крошках)
Уровень доступа к MailBox
- Минимальный уровень доступа к MailBox
Уровень доступа к ПУ MailBox
- Минимальный уровень доступа к панели управления статическими страницами
Максимальное число папок
- Максимальное кол-во папок, которые может создать пользователь (без учета системных)
Использовать систему пользователей
- Совместимость с модулем User System, которая подсвечивает выводимых пользователей, ссылкой на их профиль
Использовать E-Mail оповещения
- При получении новых непрочитанных сообщений, пользователю будет отправлено E-Mail сообщение
Использовать SSL для SMTP
- Если почтовый SMTP сервер использует SSL соединение, то необходимо включить данную настройку. Остальные настройки тут
Папок на страницу
- Кол-во выводимых папок на одну страницу
Переписок на страницу
- Кол-во выводимых переписок на одну страницу
Ответов на страницу
- Кол-во выводимых ответов в переписке на одну страницу
Папок на странице управления
- Кол-во выводимых папок на одну страницу в пользовательском управлении папками
Иконок на странице ПУ
- Кол-во выводимых иконок на одну страницу в панели управления
