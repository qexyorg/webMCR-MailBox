Установка - Шаг 1

Название модуля
- Название модуля MailBox (Отображается в заголовке страницы и хлебных крошках)
Использовать систему пользователей
- Функция добавляет дополнительный функционал при использовании модуля "User System"
Подтверждение по E-Mail
- При получении новых писем, пользователь будет получать оповещение на свой E-Mail
SSL для почтового сервера
- При использовании SSL на SMTP сервере, необходимо включить данную настройку
Максимальное число папок
- Максимально допустимое кол-во папок, создаваемых пользователем(без учета системных)
