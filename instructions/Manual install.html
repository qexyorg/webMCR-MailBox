Ручная установка

Внимание! Ручную установку стоит использовать только в том случае, если файлы .htaccess и instruments/menu_items.php были изменены

В противном случае, воспользуйтесь автоматической установкой

Открываем файл .htaccess и после строчки

RewriteRule ^go/([^/]+)/?$ index.php?mode=$1 [L,NE]

Добавляем

RewriteRule ^go/mailbox/([^/]+)/?$ index.php?mode=mailbox&do=$1 [L,NE]
RewriteRule ^go/mailbox/message/([^/]+)/?$ index.php?mode=mailbox&do=message&mid=$1 [L,NE]
RewriteRule ^go/mailbox/([^/]+)/([0-9]+)/?$ index.php?mode=mailbox&do=$1&pid=$2 [L,NE]

Открываем instruments/menu_items.php и перед строчкой

'admin' => array (

Добавляем

'qexy_mailbox' => array (

'name' => '<i class="icon-envelope"></i>',
'url' => '',
'parent_id' => -1,
'lvl' => 1,
'permission' => -1,
'active' => false,
'inner_html' => '',
),
'q_send' => array (

'name' => 'Написать',
'url' => ($config['rewrite'])? 'go/mailbox/send' : '?mode=mailbox&do=send',
